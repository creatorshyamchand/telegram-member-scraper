🛠️ Installation & Setup

1. Update & Upgrade Termux

```bash

pkg install python git -y
```

2. Install Python Dependencies

```bash

```


🔧 Troubleshooting

Common Issues & Solutions

1. Authentication Failed:

```bash
rm -rf .session_files/
python scraper.py
```

2. Module Not Found:

```bash
pip install --force-reinstall telethon
```

3. Rate Limit Error:

· Wait 5-10 minutes
· Use different account
· Reduce scraping speed

4. Channel Access Error:

· Ensure you're admin in both channels
· Check channel privacy settings
· Verify channel usernames
